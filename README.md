Stock Market Data Visualization using Power BI


 **Project Overview**

This project presents an interactive Stock Market Data Visualization Dashboard built using Power BI. The dashboard analyzes historical stock data to identify trends, price movement patterns, trading volumes, and key financial insights.

The goal of this project is to demonstrate strong data analysis, visualization, and business intelligence skills using Power BI.

 **Objectives**

Analyze historical stock price data

Identify price trends (Open, Close, High, Low)

Examine trading volume patterns

Visualize moving averages

Detect volatility patterns

Provide actionable financial insights
